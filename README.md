## 📋 About The Project

The official website of **Aeroclub NITTE** - a leading force in drone technology and aviation innovation. Aero Club, Nitte aims to foster technical excellence, build a vibrant aviation community, and expand its global influence through RC aeromodelling, workshops, competitions, and collaborative opportunities.

### 🎯 Vision
To become a leading force in drone technology and innovation, fostering technical excellence, building a vibrant aviation community, and expanding global influence through RC aeromodelling workshops, outreach programs, and collaboration.

### 🎖️ Mission
To share knowledge and expertise while nurturing a vibrant and influential aviation community, underpinned by our steadfast commitment to excellence as we continually inspire and propel our members to achieve remarkable success in the aviation realm.

## ✨ Features

- **🏠 Home Page**: Dynamic landing page with hero section and club introduction
- **🚀 Projects Showcase**: Display of innovative drone and aeromodelling projects
- **👨‍🏫 Mentors Section**: Meet the pioneers and mentors guiding the club
- **📅 Events Page**: Information about upcoming and past events, workshops, and competitions
- **👥 Team Page**: Multi-batch team structure (2020-2024 through 2024-2028)
- **🖼️ Gallery**: Visual showcase of club activities, events, and achievements
- **🏆 Achievements**: Highlight of awards and recognitions from IIT competitions and other prestigious events
- **📞 Contact Form**: Easy communication channel for inquiries and collaboration
- **📱 Responsive Design**: Fully mobile-friendly and adaptive across all devices
- **🎨 Modern UI/UX**: Clean, professional design with smooth animations

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic text editor (VS Code, Sublime Text, etc.) for modifications
- Optional: Local web server for development

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/Aeroclub-WEBSITE.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Aeroclub-WEBSITE
   ```

3. **Open with a local server**
   
   Using Python:
   ```bash
   # Python 3
   python -m http.server 8000
   ```
   
   Using PHP:
   ```bash
   php -S localhost:8000
   ```
   
   Using Node.js (with live-server):
   ```bash
   npx live-server
   ```

4. **Access the website**
   
   Open your browser and navigate to:
   ```
   http://localhost:8000
   ```

### Direct Usage

Alternatively, simply open `index.html` directly in your web browser for a quick preview.

## 🎨 Customization

### Updating Content

1. **Images**: Replace images in the `assets/img/` directory
2. **Styling**: Modify `assets/css/main.css` for custom styles
3. **Content**: Edit HTML files directly to update text and structure
4. **Colors**: Update color schemes in the CSS file

### Adding New Sections

1. Create new HTML files in the root directory
2. Add navigation links in the header section
3. Follow existing structure for consistency
4. Update sitemap if applicable

## 🌐 Deployment

### GitHub Pages

1. Push your code to GitHub
2. Go to repository Settings → Pages
3. Select branch (usually `main` or `master`)
4. Save and wait for deployment

### Custom Domain

The project includes a `CNAME` file for custom domain configuration. Update it with your domain name.
